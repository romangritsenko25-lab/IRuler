## 🛠️ Установка на GitHub Pages

### Шаг 1: Создайте репозиторий
1. Войдите в GitHub (https://github.com)
2. Нажмите кнопку **"New"** (зелёная кнопка справа вверху)
3. Введите название, например: `yandex-partners-site`
4. Выберите **Public**
5. Нажмите **"Create repository"**

### Шаг 2: Загрузите файл
1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите файл `index.html` в окно
3. Внизу нажмите **"Commit changes"**

### Шаг 3: Включите GitHub Pages
1. Перейдите в **Settings** (шестерёнка справа вверху)
2. Слева в меню найдите **Pages**
3. В разделе **Source** выберите:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажмите **Save**
5. Подождите 1-2 минуты
6. Обновите страницу - появится ссылка на ваш сайт! 🎉

## ✏️ Редактирование

### Онлайн (прямо в GitHub):
1. Откройте файл `index.html`
2. Нажмите иконку **карандаша** (Edit)
3. Внесите изменения
4. Нажмите **"Commit changes"**
5. Подождите 30 секунд - изменения появятся на сайте

### Локально (на компьютере):
1. Скачайте репозиторий: **Code** → **Download ZIP**
2. Редактируйте `index.html` в любом редакторе
3. Загрузите обратно через **Upload files**

### Цвета
Найдите секцию `:root` и измените:
```css

  --yellow: #FFD700;  /* Основной жёлтый */

  --gray: #1A1A1A;    /* Вторичный фон */

}
```
